JavaSE内部类与异常处理

发表于 2021-11-09

JavaSE

基础

super

  1. super调用父类构造器，必须在构造方法的第一个

    public class Person() {
    public Person() {
        System.out.println("Person执行了");
    }
    protected String name = "test";
    
    //私有的东西无法继承
    public void print() {
        System.out.println("Person");
    }
}
    public class Student extends Person {
    public Student() {
        //隐藏代码：调用父类的无参构造
        super();
        System.out.println("Student无参构造了");
        private String name = "test2";
        public void print() {
            System.out.println("Student");
        }
        public void test1() {
            print(); //Student
            this.print(); //Student
            super.print(): //Person
        }
    }
}

  2. super只能出现在子类的方法或构造方法中！

  3. super和this不能同时调用构造方法

接口

  1. 使用接口进行约束

  2. 定义一些方法，让不同人实现

  3. 接口中所有定义的方法都是抽象的 public abstract

    public interface UserService {
    int Age = 99;
    //所有定义的方法都是抽象的
    void add(String name);
    void delete(String name);
}

  4. 必须重写接口中的所有方法。

  5. 接口不能被实例化，且无构造方法

  6. 使用public static final不能被继承

  7. 使用implement可以实现多个接口

    public class UserServiceImpl implements UserService, User {
    public void add(String name){};
    public void delete(String name){};
}

N种内部类

  1. 内部类

    public class Outer {
    private int ID;
    public void out(){
        System.out.println("外部类方法");
    }
    public class Inner{
        public void in(){
            System.out.println("内部类方法");
        }
    }
}
    import com.Outer;

public class Application {
    
    public static void main(String[] args) {
        Outer outer = new Outer();
        //通过外部类来实例化内部类
        outer.new Inner();
    }
}

    内部类可以访问外部类的私有属性

  2. 静态内部类

    public class Outer {
    private int ID;
    public void out(){
        System.out.println("外部类方法");
    }
    public static class Inner{
        public void in(){
            System.out.println("内部类方法");
        }
    }
}

    静态内部类无法访问外部类私有属性，因为static会先初始化，此时还未初始化外部类的私有属性

  3. 写在外部类外的内部类

    public class Outer {
    private int ID;
    public void out(){
        System.out.println("外部类方法");
    }
}
class Inner{
    public void in(){
         System.out.println("内部类方法");
    }
}

    一个文件中只能有一个public类。

  4. 匿名内部类

    public class Test {
    public static void main(String[] args) {
        new UserService{
            @Override
            public void hello(){}
        }
    }
}

interface UserServices{
    void hello();
}

异常处理

  1. java异常类型

  1. 异常处理五个关键字

    • try
    • catch
    • finally
    • throw
    • throws

  2. 使用方法

    public class Test {
    public static void main(String[] args) {
        int a = 1;
        int b = 0;
        try {  //try监控区域
            System.out.println(a/b);
        }
        //catch想要捕获的异常类型
        catch(Error e){ System.out.println("Error");}
        catch(Exception e){ System.out.println("Exceotion");}
        catch(Throwable t){ System.out.println("Throwable");}
        finally { //处理善后工作，也可以不写
            System.out.println("finally");
        }
    }
}
    public class Test {
    public static void main(String[] args) {
        int a = 1;
        int b = 0;
        
        // Ctrl + Alt +T
        try {  //try监控区域
            System.out.println(a/b);
        }
        //catch想要捕获的异常类型
        catch(Exception e){ e.printStackTrace;}
        finally { //处理善后工作，也可以不写
            System.out.println("finally");
        }
    }
}
    public class Test {
    public static void main(String[] args) {
        new Test().test(1, 0);
    }
    
    public void test(int a, int b){
        if(b==0)  //throw
            throw new ArithmeticException(); // 主动的抛出异常
    }
}

    假设这方法中，处理不了这个异常，则在方法上抛出异常

    public class Test {
    public static void main(String[] args) {
        new Test().test(1, 0);
    }
    
    public void test(int a, int b) throws ArithmeticException{
        if(b==0)  //throw
            throw new ArithmeticException(); // 主动的抛出异常
    }
}
